# Simple Calculator

A simple calculator built using **HTML**, **CSS**, and **JavaScript**.

## Features
- Add, subtract, multiply, divide
- Simple and clean UI
- Works on all modern browsers
- DAark mode toggle
- Responsive design(works on mobileand desktop)
- keboard support for number and operator keys
